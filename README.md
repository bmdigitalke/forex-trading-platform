# Forex Trading Platform

## Overview
This project is a Forex Trading Platform that allows users to trade foreign currencies in real-time. It includes features such as real-time price updates, trade execution, and comprehensive analytics.

## Features
- Real-time price data
- Easy trading interface
- Advanced charting tools
- User account management

## Installation
1. Clone the repository:  
   `git clone https://github.com/bmdigitalke/forex-trading-platform.git`
2. Navigate to the project folder:  
   `cd forex-trading-platform`
3. Install dependencies:  
   `npm install` 

## Usage
To start the application, run:  
`npm start`
